## 1. 概要

### 1.1 プロジェクト概要

エンジニア向けのモダンなポートフォリオサイトスターターキットです。個人開発のWebアプリ・モバイルアプリを紹介するために設計されています。

### 1.2 設計思想

| 項目 | 説明 |
|------|------|
| テーマ | ダークテーマベース |
| スタイル | モダン・ミニマル |
| エフェクト | グラスモーフィズム、グラデーション |
| アプローチ | モバイルファースト・レスポンシブ |

### 1.3 技術スタック

| カテゴリ | 技術 |
|---------|------|
| フレームワーク | Next.js 14 (App Router) |
| 言語 | TypeScript |
| スタイリング | Tailwind CSS |
| コンテンツ | MDX |
| アイコン | Lucide React |
| フォント | Google Fonts (Noto Sans JP, Zen Kaku Gothic New, JetBrains Mono) |

### 2.4 エフェクト

#### グラスモーフィズム（.glass）

半透明の背景とブラー効果を組み合わせたモダンなスタイル。

```css
.glass {
  background-color: rgba(255, 255, 255, 0.05);
  backdrop-filter: blur(12px);
  border: 1px solid rgba(255, 255, 255, 0.1);
}
```

**適用箇所:** Header、カード、バッジ、ボタン

#### テキストグラデーション（.text-gradient）

Primary から Accent へのグラデーションテキスト。

```css
.text-gradient {
  background: linear-gradient(to right, #38bdf8, #f97316);
  -webkit-background-clip: text;
  color: transparent;
}
```

**適用箇所:** ロゴ、ヒーロー見出し、統計数値

#### 背景グラデーション（.bg-gradient-mesh）

ページ背景に使用するメッシュグラデーション。

```css
.bg-gradient-mesh {
  background:
    radial-gradient(ellipse at 20% 0%, rgba(14, 165, 233, 0.15) 0%, transparent 50%),
    radial-gradient(ellipse at 80% 100%, rgba(249, 115, 22, 0.1) 0%, transparent 50%),
    rgb(10, 10, 12);
}
```

#### グリッドパターン（.bg-grid）

背景に重ねるグリッドパターン。

```css
.bg-grid {
  background-image:
    linear-gradient(rgba(255, 255, 255, 0.03) 1px, transparent 1px),
    linear-gradient(90deg, rgba(255, 255, 255, 0.03) 1px, transparent 1px);
  background-size: 60px 60px;
}
```

#### カードホバー（.card-hover）

カードにホバーした際のエフェクト。

```css
.card-hover {
  transition: all 0.3s ease-out;
}
.card-hover:hover {
  transform: translateY(-4px);
  box-shadow: 0 20px 25px -5px rgba(14, 165, 233, 0.1);
}
```

### 2.5 アニメーション

| 名前 | 効果 | 時間 | イージング |

| `.animation-delay-300` | 300ms |
| `.animation-delay-400` | 400ms |
| `.animation-delay-500` | 500ms |

### 5.1 Header

**ファイル:** `src/components/Header.tsx`

#### 概要

固定配置のナビゲーションヘッダー。グラスモーフィズムスタイルを適用。

#### 仕様

| 項目 | 値 |
|------|-----|
| 配置 | `fixed top-0 left-0 right-0` |
| Z-index | `z-50` |
| 高さ | `h-16` (64px) |
| スタイル | `.glass` |

#### デスクトップナビゲーション（md以上）

- 横並びのリンク
- Pill型（`rounded-full`）
- アクティブ状態: `bg-white/10 text-white`
- 非アクティブ状態: `text-gray-400 hover:text-white hover:bg-white/5`

#### モバイルナビゲーション（md未満）

- ハンバーガーメニューボタン
- 開閉式のドロップダウンメニュー
- アイコン: Menu（開）/ X（閉）

#### 状態管理

```typescript
const [isMenuOpen, setIsMenuOpen] = useState(false);
```

### 5.3 ProjectCard

**ファイル:** `src/components/ProjectCard.tsx`

#### 概要

プロジェクトを表示するカードコンポーネント。通常表示と注目表示の2種類がある。

#### Props

| 名前 | 型 | 必須 | デフォルト | 説明 |
|------|-----|------|-----------|------|
| `project` | `Project` | ✓ | - | プロジェクトデータ |
| `featured` | `boolean` | - | `false` | 注目表示フラグ |

#### 表示バリエーション

| 種類 | グリッド | レイアウト | 技術バッジ数 |
|------|---------|-----------|-------------|
| 通常 | 1列幅 | 縦並び（サムネイル上、コンテンツ下） | 最大4件 |
| featured | 2列幅 | 横並び（サムネイル左、コンテンツ右） | 最大6件 |

#### 構成要素

1. **サムネイル**: アスペクト比16:9、ホバー時スケール1.05
2. **カテゴリーバッジ**: 左上に配置
3. **タイトル**: ホバー時 primary-400 に変化
4. **説明文**: 2行でクリップ（`line-clamp-2`）
5. **技術バッジ**: TechBadge コンポーネント使用
6. **リンク**: Demo（primary）、Code（gray）

### 5.4 TechBadge

**ファイル:** `src/components/TechBadge.tsx`

#### 概要

技術名を表示するバッジコンポーネント。技術名に応じた色分けを行う。

#### Props

| 名前 | 型 | 必須 | デフォルト | 説明 |
|------|-----|------|-----------|------|
| `name` | `string` | ✓ | - | 技術名 |
| `size` | `'sm' \| 'md'` | - | `'sm'` | サイズ |

#### サイズ

| サイズ | パディング | フォントサイズ |
|--------|-----------|---------------|
| `sm` | `px-2.5 py-0.5` | `text-xs` |
| `md` | `px-3 py-1` | `text-sm` |

#### 色分けルール

技術名に応じて背景色・テキスト色・ボーダー色を設定。

| 技術 | 色系統 |
|------|--------|
| React, React Native, Go | Cyan |
| Vue.js, Node.js, MongoDB | Green/Emerald |
| TypeScript, PostgreSQL, Docker, GCP | Blue |
| JavaScript, Python, Firebase | Yellow/Amber |
| Swift, Rust, AWS, MySQL | Orange |
| Kotlin | Purple |
| Flutter | Sky |
| Next.js, Vercel | White |
| Redis | Red |
| Tailwind CSS | Teal |
| その他 | Gray（フォールバック） |

## 6. インタラクション

### 6.1 ホバーエフェクト

#### リンク

| 要素 | 効果 |
|------|------|
| ナビゲーションリンク | テキスト色変化 + 背景色変化 |
| テキストリンク | テキスト色変化（primary-400 → primary-300） |
| ソーシャルリンク | 色変化 + 背景色変化 |

#### カード

| 効果 | 値 |
|------|-----|
| 上昇 | `translateY(-4px)` |
| シャドウ | `shadow-xl shadow-primary-500/10` |
| トランジション | `0.3s ease-out` |

#### サムネイル画像

| 効果 | 値 |
|------|-----|
| スケール | `scale(1.05)` |
| トランジション | `0.5s` |

#### ボタン

| 種類 | ホバー効果 |
|------|-----------|
| プライマリ | `bg-primary-500` → `bg-primary-600` |
| セカンダリ（glass） | `bg-white/5` → `bg-white/10` |

### 6.2 フォーカス状態

アクセシビリティのため、フォーカス可能な要素にはフォーカスリングを適用。

### 6.3 アニメーション適用箇所

#### ホームページ

| 要素 | アニメーション | 遅延 |
|------|--------------|------|
| ウェルカムバッジ | `fade-in` | 0ms |
| メイン見出し | `slide-up` | 0ms |
| 説明文 | `slide-up` | 100ms |
| CTAボタン | `slide-up` | 200ms |

### 6.4 トランジション

すべてのインタラクティブ要素に `transition-all` または `transition-colors` を適用。

| 用途 | duration |
|------|----------|
| 色変化 | `duration-200` |
| 全体変化 | `duration-300` |
| 画像スケール | `duration-500` |
